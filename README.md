# 🌐 Modern Developer Portfolio

## **📌 Overview**
A modern and fully responsive portfolio website built to showcase my projects, skills, and experience. The portfolio features a clean UI, smooth navigation, and an interactive user experience with light and dark mode support. Visitors can explore my work and contact me directly through the integrated email functionality.
